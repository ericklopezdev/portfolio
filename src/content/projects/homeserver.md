# Home Server Automation

A fully automated home server environment featuring media streaming, monitoring, and backup solutions with infrastructure as code.

## Features

- Automated deployment with Ansible
- Media server with Plex/Jellyfin
- Comprehensive monitoring stack
- Automated backups to cloud storage
- VPN access with WireGuard
- Docker container orchestration
- Energy-efficient setup

## Technologies

- Infrastructure: Ansible, Terraform
- Containers: Docker, Docker Compose
- Monitoring: Prometheus, Grafana, Node Exporter
- Media: Plex, Sonarr, Radarr
- Storage: MergerFS, SnapRAID
- OS: Ubuntu Server, Proxmox
