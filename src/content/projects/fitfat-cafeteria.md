# Sistema de Gestión FitFat Cafeteria

Sistema de gestión para un restaurante/cafetería en mi localidad, facilitando el control de pedidos, inventario y atención al cliente.

## Características

- Sistema de pedidos en línea
- Seguimiento de pedidos en tiempo real
- Gestión de inventario con alertas
- Programa de fidelización de clientes
- Programación y gestión de personal
- Analíticas de ventas y reportes
- Aplicación móvil para clientes
- Integración con pasarelas de pago

## Tecnologías

- Frontend: React
- Backend: Node.js, Express
- Base de datos: Postgres
- Autenticación: JWT
- Hosting: Railway
